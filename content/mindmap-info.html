📖 วิธีใช้งาน

MindMap ใช้จัดความคิดให้เห็นความสัมพันธ์ของหัวข้อ เนื้อหา และข้อมูลประกอบในหน้าเดียว

เลือกใช้ 📥 Import เพื่อเปิด JSON เดิมหรือ Project ZIP, 📤 Export เพื่อเก็บโปรเจกต์พร้อมภาพต้นฉบับ, 💾 HTML เพื่อบันทึกหน้า MindMap ไฟล์เดียวพร้อมภาพย่อ หรือ 📸 PNG เพื่อบันทึกภาพ

🧩 ใช้งานยังไง?
  • ดับเบิลคลิกพื้นที่ว่างเพื่อสร้าง node
  • แก้ไขหัวข้อและรายละเอียดในกล่องได้โดยตรง
  • ลากปุ่ม + รอบ node เพื่อเชื่อม, แตกแขนง หรือสร้าง node ใหม่
  • ชี้และคลิกเส้นเชื่อมเพื่อแทรกจุดเชื่อมแล้วสร้าง node ใหม่; กดค้างแล้วลากจากเส้นเพื่อวาง node ณ จุดที่ปล่อย
  • Alt + คลิกเส้นเชื่อม เพื่อวนหัวลูกศร: ไม่มี → ไปข้างหน้า → ย้อนกลับ → สองหัว ↔
  • ใช้ 🔎 Find ค้นหาหัวข้อ, รายละเอียด, และ Note; ข้อความที่พบในหัวข้อ/รายละเอียดจะถูกเน้นสี ส่วนที่พบใน Note จะทำให้ปุ่ม 📝 ของ node แสดงและถูกเน้น โดยไม่แสดงข้อความ Note บน Canvas; ในหน้าต่าง Note มี 🔎 ค้นหาในบันทึก ด้านบนเสมอ สำหรับเลือกคำที่พบทีละตำแหน่ง
  • Shift + คลิก หรือ ลากพื้นที่ว่างเพื่อเลือกหลาย node และหลายภาพพร้อมกัน; ลากวัตถุชิ้นใดในกลุ่มเพื่อย้ายทั้งกลุ่ม โดย 🧲 Snap รักษาระยะห่างเดิมของทุกชิ้น
  • กด Alt ค้างพร้อมลากเพื่อคัดลอก Node, รูป, Line หรือ Text ลอยได้ทันที; หากลาก Node/รูปที่อยู่ในชุดเลือก จะคัดลอกทั้งชุด และหากมีชุดเดิมอยู่จะรวมชิ้นที่ลากเพิ่มเข้าไปด้วย; ชุดสำเนาวางตรงตำแหน่งที่ปล่อยเมาส์
  • คลิกขวาหรือกดเมาส์กลางค้างเพื่อเลื่อนมุมมอง
🧭 ทำยังไง?
  • ใช้เมาส์เพื่อ Zoom เข้า-ออก ได้
  • Theme วน Sepia → Dark → Sepia with Dark Canvas
  • ▦ Grid แสดงตาราง และ 🧲 Snap ล็อกกึ่งกลาง node ลงกริด
  • กดตัวเลขเปอร์เซ็นต์มุมซ้ายล่างเพื่อกลับสู่ 100%
  • กด Alt + R เพื่อ Fit graph; กดซ้ำเพื่อกลับมุมเดิม หรือหลังเลื่อนมุมมองให้กลับระดับ Zoom เดิมที่ตำแหน่งเมาส์
  • ชี้หรือเลือก node เพื่อเห็นป้ายความสัมพันธ์ของทุกเส้นรอบ node เช่น → +; คลิกป้ายเพื่อไปยัง node อีกฝั่ง, ดับเบิลคลิกเพื่อพิมพ์หรือแก้คำ และลากเพื่อย้ายตำแหน่งบนเส้น ลูกศรจะชี้ตามทิศที่เส้นออกจาก node
  • เลือกหลาย node เพื่อใช้ T/C/B-Align, L/C/R-Align และ Dist
✨ นอกจาก node ที่เห็นแล้วมีอะไรอีก?
  • กด 🖼️ IMG หรือวางไฟล์ภาพลงบน Canvas ได้; 🌄 ภาพ ใช้ซ่อน/แสดงภาพชั่วคราวเพื่อลดภาระการแสดงผลโดยไม่ลบข้อมูล
  • คลิกภาพเพื่อย้าย, ลากจุดมุมเพื่อปรับขนาด; กด Shift เพื่อรักษาสัดส่วน, Ctrl เพื่อรักษาจุดศูนย์กลาง, หรือกดทั้งคู่พร้อมกัน
  • ดับเบิลคลิกภาพ หรือกด i ข้างภาพ เพื่อดูภาพเต็มและบันทึกคำบรรยาย, แหล่งที่มา, และบันทึกเพิ่มเติม
  • กด 📝 ที่มุมซ้ายล่างของ node เพื่อเปิดพื้นที่บันทึกข้อความหลายบรรทัด
  • กด │ เพื่อเพิ่มเส้นแบ่งแนวตั้ง; เลือกเส้นแล้วลากปลายเส้นเพื่อกำหนดมุม, กด Shift เพื่อล็อกแนวนอนหรือแนวตั้ง, ใช้ 🧲 Snap เพื่อล็อกปลายเส้นเข้ากริด และกด Delete เพื่อลบ
  • กด T เพื่อสร้างหัวข้อช่วงเวลา พร้อมคำอธิบายบรรทัดเล็กด้านล่าง; พิมพ์แทนข้อความเริ่มต้นได้ทันที, ดับเบิลคลิกเพื่อแก้ไขอีกครั้ง, กด Shift + Enter เพื่อขึ้นบรรทัดใหม่, ลากเพื่อย้าย และใช้ A+/A- หรือแถบสีเพื่อปรับรูปแบบ; ระหว่างลากกด Shift เพื่อล็อกการเคลื่อนที่แนวนอนหรือแนวตั้ง
  • กด 🟨 Area เพื่อเข้าโหมดวาดพื้นหลัง: ลากพื้นที่ว่างเพื่อสร้าง, ลากพื้นที่เพื่อย้าย, ใช้จุดรอบกรอบเพื่อปรับขนาด และเลือกสีจากแถบสี; กด Esc หรือ 🟨 Area ซ้ำเพื่อออกจากโหมด แล้วพื้นที่สีจะคลิกทะลุได้ตามปกติ
⌨️ Hotkeys
  • Alt + คลิกเส้นเชื่อม วนหัวลูกศร
  • Alt + ลาก คัดลอก Node/รูป, Line หรือ Text ลอย; Node/รูปที่มีชุดเลือกจะคัดลอกทั้งชุด
  • Shift + ลาก Floating Text ล็อกแนวนอนหรือแนวตั้ง
  • Alt + S 💾 HTML
  • Alt + Z ↩️ Undo / Alt + Y ↪️ Redo
  • Alt + B คัดลอก Bookmark
  • Alt + R Fit graph / กลับมุมมองเดิม
  • Alt + U, J, M สำหรับ T/C/B-Align
  • Alt + I, O, P สำหรับ L/C/R-Align
  • Alt + K / L สำหรับ ↔ Dist / ↕ Dist
  • Alt + 1 ถึง ' เลือกสีกรอบ, Alt + \ ล้างสีกรอบ
  • Delete ลบ node, รูป, Line, Text ลอย หรือ Area ที่เลือก (นอกช่องพิมพ์)
  • Esc ออกจากโหมด Area หรือยกเลิกการลาก Area; Tab / Shift + Tab สลับหัวข้อและรายละเอียด; ใน 🔎 Find ใช้ Enter / Shift + Enter เปลี่ยนผลค้นหา
