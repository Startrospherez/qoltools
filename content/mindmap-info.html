🌿 แนะนำเบื้องต้น

MindMap ใช้จัดความคิดให้เห็นความสัมพันธ์ของหัวข้อ เส้นทาง เนื้อหา ภาพ และข้อมูลประกอบในกระดานเดียว เหมาะทั้งงานเล็กและแผนผังระยะยาว

📥 Import เปิด JSON เดิมหรือ Project ZIP, 📤 Export เก็บโปรเจกต์เป็น ZIP พร้อมภาพต้นฉบับ, 💾 HTML สร้างไฟล์เปิดดูได้ทันทีพร้อมภาพย่อ และ 📸 PNG บันทึกเป็นภาพ

🌱 วิธีใช้งาน Basic
  • ดับเบิลคลิกพื้นที่ว่างเพื่อสร้าง Node แล้วแก้หัวข้อและรายละเอียดในกล่องได้โดยตรง
  • ลากปุ่ม + รอบ Node เพื่อเชื่อม แตกแขนง หรือสร้าง Node ใหม่
  • ลาก Node, รูป, │ Line, T Text ลอย หรือ 🟨 Area เพื่อจัดตำแหน่ง; กด Shift ระหว่างลาก Text ลอยเพื่อล็อกแนวนอนหรือแนวตั้ง
  • คลิกวัตถุเพื่อเลือก, Shift + คลิกเพื่อเพิ่มหรือลดจากชุดเลือก และลากพื้นที่ว่างเพื่อคลุมเลือกหลายชิ้น
  • ▦ Grid แสดงตาราง, 🧲 Snap ล็อกตำแหน่ง และเมาส์กลางหรือคลิกขวาค้างใช้เลื่อนกระดาน
  • ใช้ล้อเมาส์ Zoom เข้า-ออก; คลิกตัวเลขเปอร์เซ็นต์มุมซ้ายล่างเพื่อกลับ 100%
🧰 วิธีใช้งาน Intermediate
  • 🔎 Find ค้นหัวข้อ รายละเอียด Note และข้อมูล Area; ผลใน Note จะเน้นปุ่ม 📝 โดยไม่เปิดข้อความบน Canvas
  • กด 📝 ของ Node หรือ Area เพื่อเก็บ Note หลายบรรทัด และใช้ช่องค้นหาภายในบันทึกเลือกคำที่พบทีละตำแหน่ง
  • 🖼️ IMG หรือวางไฟล์ภาพลง Canvas; ลากมุมเพื่อปรับขนาด, Shift รักษาสัดส่วน, Ctrl รักษาศูนย์กลาง และดับเบิลคลิกเพื่อดูภาพเต็มพร้อมคำบรรยาย/แหล่งที่มา
  • เลือกหลายวัตถุแล้วใช้ T/C/B-Align, L/C/R-Align และ Dist; การลาก ลบ จัดแนว และปรับขนาดแบบกลุ่มย้อนกลับได้ด้วย Undo ครั้งเดียวต่อกิจกรรม
  • กด Alt ค้างแล้วลากเพื่อทำสำเนาวัตถุหรือชุดที่เลือก และวางสำเนาตรงตำแหน่งปล่อยเมาส์
  • Theme วน Sepia → Dark → Sepia with Dark Canvas; 🎨 เลือกตระกูลสีสำหรับการลงสีครั้งถัดไป โดยสีที่ลงไว้แล้วไม่เปลี่ยนตาม
  • กด Alt + R เพื่อ Fit graph; กดซ้ำเพื่อกลับมุมมองเดิม หรือกลับระดับ Zoom เดิม ณ ตำแหน่งเมาส์หลังเลื่อนกระดาน
🛠️ วิธีใช้งาน Advanced
  • ชี้หรือเลือก Node เพื่อเห็นป้ายความสัมพันธ์รอบ Node; คลิกป้ายเพื่อไปปลายทาง, ดับเบิลคลิกเพื่อแก้คำ และลากเพื่อย้ายป้ายบนเส้น
  • คลิกเส้นเชื่อมเพื่อสร้าง Node จากจุดนั้น หรือกดค้างแล้วลากออกไปเพื่อกำหนดตำแหน่ง Node ใหม่
  • Alt + คลิกซ้ายเส้นเชื่อม วนหัวลูกศร: ไม่มี → ไปข้างหน้า → ย้อนกลับ → สองหัว; Alt + เมาส์กลางบนเส้น วนเส้นทึบ → เส้นจุด → เส้นประ
  • │ Line เป็นเส้นแบ่งอิสระ: ลากปลายเพื่อกำหนดมุม, กด Shift เพื่อล็อกแนวนอน/แนวตั้ง และใช้ Snap ล็อกปลายลงกริด
  • T Text ลอย มีชื่อและ Description, ดับเบิลคลิกเพื่อแก้, Shift + Enter เพื่อขึ้นบรรทัดใหม่ และใช้ A+/A− ปรับขนาด
  • 🟨 Area เปิดโหมดวาดพื้นหลัง: ลากเพื่อสร้าง/ย้าย, ใช้จุดรอบกรอบปรับขนาด, ใส่ชื่อ Description และ Note; ออกจากโหมดด้วย Esc แล้วพื้น Area จะคลิกทะลุได้
  • A+/A− กดหนึ่งครั้งปรับหนึ่งระดับ หรือกดค้างเพื่อปรับต่อเนื่อง; ใช้กับ Node, Text ลอย และข้อความ Area
⌨️ Hotkeys
  • Alt + ลาก — คัดลอกวัตถุหรือชุดที่เลือก
  • Alt + คลิกซ้ายเส้น — วนหัวลูกศร; Alt + เมาส์กลางบนเส้น — วนรูปแบบเส้น
  • Alt + S — 💾 HTML; Alt + B — คัดลอก Bookmark; Alt + R — Fit/กลับมุมมอง
  • Alt + Z — ↩️ Undo; Alt + Y — ↪️ Redo
  • Alt + U/J/M — T/C/B-Align; Alt + I/O/P — L/C/R-Align
  • Alt + K/L — กระจายแนวนอน/แนวตั้ง
  • Alt + 1 ถึง ' — เลือกสีกรอบ; Alt + \ — ล้างสีกรอบ
  • Delete — ลบชุดที่เลือก; Esc — ปิดหน้าต่าง/ออกโหมด Area; Tab/Shift + Tab — สลับหัวข้อกับรายละเอียด
  • ใน 🔎 Find ใช้ Enter/Shift + Enter เพื่อไปผลถัดไป/ก่อนหน้า
