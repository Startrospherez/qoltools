# MindMap AI Stress Fixture — Design

## Purpose

Create a local, importable MindMap Project ZIP that exercises a large graph
without mixing synthetic material with the user's Buddhist study data.

## Chosen approach

Use a deterministic, image-free 5 × 5 hierarchy. Each of the 25 chapters has
one root, three sections, and 12 leaves per section: exactly 1,000 Nodes and
975 connectors. A small set of floating headings/dividers and repeated Notes
also exercise those persisted data types.

The ZIP follows the existing Project format (`manifest.json` + `mindmap.json`)
and declares no image assets. It is generated by
`generate-mindmap-stress-fixture.py`, so it can be recreated exactly.

## Deliberate difference from a human map

The synthetic map is uniformly spaced, exactly three levels deep, and uses
regular names/descriptions. Human-authored maps normally have uneven density,
mixed hierarchy depth, irregular cross-links, varied text lengths, and more
notes/images. This fixture is therefore appropriate for graph rendering,
selection, drag, Zoom/Fit, Find, Undo/Redo, Import, and Export checks, but it
does not replace testing with the user's real map.

## Storage and verification

The generated ZIP stays at `test-fixtures/mindmap-ai-stress-1000.zip` and is
ignored by Git because it is a regenerable test artifact. The generator and
this design document are tracked. Verify the ZIP contents, manifest, counts,
endpoint validity, and Import manually in MindMap.
